# FUN2020block

私が在籍する大学から出された課題「ブロック崩しの作成」に対する私の作品です。

## ゲームの説明
1. Chargeボタンを押して離すとモンスターが弾を出します。
2. Chargeボタンでパワーをためれば弾の軌道はより水平に近づきます。
3. ランダムな高さ・傾斜で現れる的に合わせて弾を発射し、的に当てましょう。
